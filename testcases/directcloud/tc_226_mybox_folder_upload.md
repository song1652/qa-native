# 마이박스 — 폴더 업로드 버튼 존재 확인

## Precondition
- 로그인 완료, 마이박스 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 폴더 업로드 버튼 또는 메뉴 항목 존재 확인

## Expected
- 폴더 업로드 버튼 또는 옵션이 표시된다
