# 설정 모달 — 알림 설정 탭/섹션 존재 확인

## Precondition
- 로그인 완료, 설정 모달(#modal-settings) 열린 상태

## Steps
1. 유효한 자격증명으로 로그인
2. 사이드바 프로필(.nav-profile) 클릭 → 설정 모달 오픈
3. 알림 설정 탭 또는 섹션 존재 확인

## Expected
- 설정 모달에 알림 설정 관련 탭 또는 섹션이 표시된다
