# 공지사항 — 페이지 로드 및 컬럼 헤더 확인

## Precondition
- 로그인 완료

## Steps
1. 유효한 자격증명으로 로그인
2. 공지사항 메뉴 클릭
3. 컬럼 헤더 확인 (제목, 구분, 날짜)

## Expected
- 공지사항 페이지가 로드된다
- 제목, 구분, 날짜 컬럼이 표시된다
