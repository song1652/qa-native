# 연락처 — 개별 등록 버튼 표시 확인

## Precondition
- 로그인 완료, 연락처 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "연락처"(li#contacts) 클릭
3. "등록" 버튼 존재 확인

## Expected
- 개별 연락처 등록 버튼이 표시된다
