# 마이박스 — 잠금된 파일에 잠금 아이콘 표시 확인

## Precondition
- 로그인 완료, 마이박스에 잠금된 파일 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 잠금된 파일 행에서 잠금 아이콘 표시 확인

## Expected
- 잠금된 파일 행에 자물쇠 아이콘이 표시된다
