# 마이박스 — 링크 생성된 파일에 링크 아이콘/배지 표시 확인

## Precondition
- 로그인 완료, 마이박스에 링크 생성된 파일 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 링크가 생성된 파일 행에서 링크 아이콘 또는 배지 표시 확인

## Expected
- 링크가 생성된 파일 행에 링크 아이콘이 표시된다
