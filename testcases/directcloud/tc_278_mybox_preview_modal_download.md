# 마이박스 — 미리보기 모달 내 다운로드 버튼 확인

## Precondition
- 로그인 완료, 미리보기 모달 열린 상태

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 파일 클릭으로 미리보기 모달 오픈
4. 모달 내 다운로드 버튼 존재 확인

## Expected
- 미리보기 모달에 다운로드 버튼이 표시된다
