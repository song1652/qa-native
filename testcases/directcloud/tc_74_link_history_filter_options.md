# Link History — 필터 select 옵션 전체/생성/수정/웹메일/업로드/링크변경 확인

## Precondition
- 로그인 완료, https://web.directcloud.jp/linkmanager 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "Link History"(li#linkmanager) 클릭
3. 필터 select 요소 옵션 확인

## Expected
- select에 "전체"(value=all) 옵션이 존재한다
- select에 "생성"(value=create) 옵션이 존재한다
- select에 "웹메일 발송"(value=webmail) 옵션이 존재한다
- select에 "업로드"(value=upload) 옵션이 존재한다
