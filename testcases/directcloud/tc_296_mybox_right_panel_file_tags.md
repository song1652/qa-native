# 마이박스 — 우측 정보 패널 태그 목록 표시 확인

## Precondition
- 로그인 완료, 마이박스에 태그가 있는 파일 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 태그가 있는 파일 선택
4. 우측 정보 패널에서 태그 목록 확인

## Expected
- 우측 정보 패널에 파일에 붙은 태그 목록이 표시된다
