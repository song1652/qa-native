# 공지사항 — 구분: 업데이트 항목 표시 확인

## Precondition
- 로그인 완료, 공지사항 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. 공지사항 메뉴 클릭
3. 구분 컬럼에서 "업데이트" 타입 공지 확인

## Expected
- 구분 컬럼에 "업데이트" 타입 공지가 표시된다
