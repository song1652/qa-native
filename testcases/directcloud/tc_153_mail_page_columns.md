# 메일 — 수신함 컬럼(수신자/제목/첨부파일/날짜) 표시 확인

## Precondition
- 로그인 완료

## Steps
1. 유효한 자격증명으로 로그인
2. "메일"(li#mail) 클릭
3. 컬럼 헤더 확인 (수신자, 제목, 첨부파일, 날짜)

## Expected
- 메일 페이지가 로드된다
- 수신자, 제목, 첨부파일, 날짜 컬럼이 표시된다
