# 연락처 — CSV 샘플 파일 다운로드 버튼 확인

## Precondition
- 로그인 완료, 연락처 CSV 업로드 모달 열린 상태

## Steps
1. 유효한 자격증명으로 로그인
2. "연락처"(li#contacts) 클릭
3. "CSV 일괄등록" 버튼 클릭
4. 샘플 CSV 다운로드 링크/버튼 존재 확인

## Expected
- 샘플 CSV 다운로드 버튼 또는 링크가 표시된다
