# 사이드바 — 로고 클릭 시 홈 이동 확인

## Precondition
- 로그인 완료

## Steps
1. 유효한 자격증명으로 로그인
2. 서브 페이지(예: 최근파일) 이동
3. 사이드바 상단 로고 클릭
4. 홈 또는 마이박스 페이지로 이동 확인

## Expected
- 로고 클릭 시 홈 또는 기본 페이지로 이동된다
