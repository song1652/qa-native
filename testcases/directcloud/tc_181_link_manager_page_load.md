# 링크 관리 — 페이지 로드 및 링크 목록 확인

## Precondition
- 로그인 완료

## Steps
1. 유효한 자격증명으로 로그인
2. "링크 관리"(li#linkmanager) 클릭
3. 링크 목록 페이지 로드 확인

## Expected
- 링크 관리 페이지가 로드된다
- 생성된 링크 목록 또는 빈 상태 메시지가 표시된다
