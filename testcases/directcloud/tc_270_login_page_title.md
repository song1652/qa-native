# 로그인 페이지 — 페이지 타이틀 및 로고 표시 확인

## Precondition
- https://web.directcloud.jp/login 접속 상태

## Steps
1. 로그인 페이지 접속
2. 페이지 타이틀 확인
3. DirectCloud 로고 표시 확인

## Expected
- 페이지 타이틀에 "DirectCloud" 또는 서비스명이 포함된다
- 로고가 표시된다
