# 메일 — 메일 작성/전송 버튼 존재 확인

## Precondition
- 로그인 완료, 메일 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "메일"(li#mail) 클릭
3. 메일 작성 버튼(새 메일 또는 편지쓰기) 존재 확인

## Expected
- 메일 작성 또는 새 메일 보내기 버튼이 표시된다
