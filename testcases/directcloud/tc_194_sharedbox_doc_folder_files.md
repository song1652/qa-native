# 공유박스 — Doc 폴더 진입 후 파일 목록 확인

## Precondition
- 로그인 완료, 공유박스에 Doc 폴더 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "공유박스"(li#sharedbox) 클릭
3. "Doc" 폴더 더블클릭
4. Doc 폴더 내 파일 목록 확인

## Expected
- Doc 폴더로 진입된다
- 폴더 내 파일 목록 또는 빈 폴더 메시지가 표시된다
