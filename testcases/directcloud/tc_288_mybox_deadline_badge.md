# 마이박스 — 기한이 설정된 파일에 기한 표시 확인

## Precondition
- 로그인 완료, 마이박스에 기한이 설정된 파일 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 기한이 설정된 파일 행에서 기한 아이콘 또는 날짜 표시 확인

## Expected
- 기한이 설정된 파일 행에 기한 관련 표시가 나타난다
