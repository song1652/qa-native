# 최근파일 — 파일 클릭 시 미리보기 오픈 확인

## Precondition
- 로그인 완료, 최근파일 페이지, 파일 최소 1개 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "최근파일"(li#recents) 클릭
3. 파일 행 클릭
4. 미리보기 모달 또는 상세 패널 오픈 확인

## Expected
- 파일 클릭 시 미리보기가 열린다
