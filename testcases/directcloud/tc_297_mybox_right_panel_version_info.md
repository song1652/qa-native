# 마이박스 — 우측 정보 패널 버전 정보 표시 확인

## Precondition
- 로그인 완료, 마이박스에 버전 이력 있는 파일 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 버전이 있는 파일 선택
4. 우측 정보 패널에서 버전 정보 확인

## Expected
- 우측 정보 패널에 현재 버전 번호 또는 버전 수가 표시된다
