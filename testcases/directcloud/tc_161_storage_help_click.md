# 스토리지 용량 도움말(?) 버튼 클릭 동작 확인

## Precondition
- 로그인 완료, 사이드바 표시 상태

## Steps
1. 유효한 자격증명으로 로그인
2. 사이드바 하단 스토리지 영역의 도움말(?) 버튼 클릭
3. 팝업 또는 툴팁 표시 확인

## Expected
- 도움말 버튼 클릭 시 스토리지 관련 안내 팝업 또는 툴팁이 표시된다
