# 알림 — 알림 벨 아이콘 클릭 시 알림 패널 오픈 확인

## Precondition
- 로그인 완료

## Steps
1. 유효한 자격증명으로 로그인
2. 상단 또는 사이드바의 알림 벨 아이콘 클릭
3. 알림 목록 패널 오픈 확인

## Expected
- 알림 벨 클릭 시 알림 패널이 열린다
- 알림 목록 또는 빈 상태 메시지가 표시된다
