# 링크 관리 — 링크 삭제 버튼 존재 확인

## Precondition
- 로그인 완료, 링크 관리 페이지, 링크 최소 1개 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "링크 관리"(li#linkmanager) 클릭
3. 링크 항목의 삭제 버튼 또는 삭제 아이콘 존재 확인

## Expected
- 링크 삭제 버튼이 표시된다
