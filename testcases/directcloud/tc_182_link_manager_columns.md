# 링크 관리 — 목록 컬럼 헤더 확인

## Precondition
- 로그인 완료, 링크 관리 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "링크 관리"(li#linkmanager) 클릭
3. 컬럼 헤더 확인 (파일명, 링크, 만료일, 다운로드 횟수 등)

## Expected
- 링크 목록 컬럼 헤더가 표시된다
