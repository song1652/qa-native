# 파일 요청 — 새 요청 생성 버튼 존재 확인

## Precondition
- 로그인 완료, 파일 요청 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "파일 요청"(li#file-requests) 클릭
3. 새 파일 요청 생성 버튼(점 3개 또는 + 버튼) 존재 확인

## Expected
- 새 파일 요청 생성 버튼이 표시된다
