# 사이드바 — 스토리지 사용량 바 표시 확인

## Precondition
- 로그인 완료

## Steps
1. 유효한 자격증명으로 로그인
2. 사이드바 하단 스토리지 영역 확인
3. 사용량 프로그레스 바 또는 수치 표시 확인

## Expected
- 사이드바에 스토리지 사용량(예: 1.2GB / 10GB)이 표시된다
- 사용량 비율을 나타내는 프로그레스 바가 표시된다
