# 검색 결과 — "추가" 버튼 표시 확인

## Precondition
- 로그인 완료, 검색 결과 1건 이상 존재

## Steps
1. 유효한 자격증명으로 로그인
2. 검색창(#inputSearch)에 검색어 입력
3. 검색 실행(#search-search 클릭 또는 Enter)
4. 검색 결과 목록에서 "추가" 버튼 존재 확인

## Expected
- 검색 결과 항목에 "추가" 버튼이 표시된다
