# 마이박스 — 파일 업로드 시 진행률 표시 확인

## Precondition
- 로그인 완료, 마이박스 페이지, 업로드 파일 준비됨

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 파일 업로드 버튼 클릭 → 파일 선택
4. 업로드 진행률 표시 확인

## Expected
- 업로드 중 진행률 표시(프로그레스 바 또는 퍼센트)가 나타난다
