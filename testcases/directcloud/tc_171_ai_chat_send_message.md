# AI 홈 — 질문 입력 후 전송 확인

## Precondition
- 로그인 완료, AI 홈 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "AI 홈"(li#aihome) 클릭
3. 질문 입력창에 텍스트 입력
4. "전송" 버튼 클릭 또는 Enter 키 입력
5. AI 응답 수신 확인

## Expected
- 질문이 전송된다
- AI 응답이 채팅창에 표시된다
