# 마이박스 — 파일 목록 컬럼(이름/크기/날짜/확장자) 표시 확인

## Precondition
- 로그인 완료, 마이박스 페이지

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 파일 목록 컬럼 헤더 확인 (이름, 크기, 날짜, 확장자 등)

## Expected
- 파일 목록에 이름, 크기, 날짜, 확장자 컬럼이 표시된다
