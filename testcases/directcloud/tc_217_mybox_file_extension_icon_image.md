# 마이박스 — 이미지 파일 썸네일 표시 확인

## Precondition
- 로그인 완료, 마이박스에 이미지 파일(JPG/PNG) 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 이미지 파일의 썸네일 표시 확인

## Expected
- 이미지 파일에 미리보기 썸네일이 표시된다
