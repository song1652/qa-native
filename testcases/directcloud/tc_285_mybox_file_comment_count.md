# 마이박스 — 파일 목록에 코멘트 수 표시 확인

## Precondition
- 로그인 완료, 마이박스에 코멘트 있는 파일 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. 코멘트가 있는 파일 행에서 코멘트 수 또는 아이콘 표시 확인

## Expected
- 파일 행에 코멘트 수 또는 코멘트 아이콘이 표시된다
