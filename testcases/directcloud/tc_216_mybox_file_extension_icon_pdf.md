# 마이박스 — PDF 파일 확장자 아이콘 표시 확인

## Precondition
- 로그인 완료, 마이박스에 PDF 파일 존재

## Steps
1. 유효한 자격증명으로 로그인
2. "마이박스"(li#mybox) 클릭
3. PDF 파일의 확장자 아이콘/썸네일 표시 확인

## Expected
- PDF 파일에 PDF 전용 아이콘이 표시된다
