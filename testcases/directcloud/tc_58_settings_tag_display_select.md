# 설정 모달 — 리스트화면 태그표시 select (표시/비표시) 확인

## Precondition
- 로그인 완료, 설정 모달(#modal-settings) 열린 상태

## Steps
1. 유효한 자격증명으로 로그인
2. 사이드바 프로필(.nav-profile) 클릭 → 설정 모달 오픈
3. 리스트화면 태그표시 select 요소 옵션 확인

## Expected
- 태그표시 select에 "표시" 옵션이 존재한다
- 태그표시 select에 "비표시" 옵션이 존재한다
