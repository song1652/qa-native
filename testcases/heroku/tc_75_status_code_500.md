# 상태 코드 500 페이지 확인

## Precondition
- https://the-internet.herokuapp.com/status_codes 접속 상태

## Steps
1. "500" 링크 클릭
2. 페이지 응답 상태 및 텍스트 확인

## Expected
- 페이지에 "500" 상태 코드 설명이 표시된다
- "This page returned a 500 status code" 텍스트가 포함된다
