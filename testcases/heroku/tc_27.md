# Challenging DOM 버튼 클릭

## Precondition
- https://the-internet.herokuapp.com/challenging_dom 접속 상태

## Steps
1. 페이지 내 버튼 요소들 확인
2. 첫 번째 버튼 클릭
3. 페이지 갱신 후 테이블 존재 확인

## Expected
- 버튼이 3개 존재한다
- 버튼 클릭 후 페이지가 정상 갱신된다
