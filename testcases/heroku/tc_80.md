# TinyMCE 에디터 기본 텍스트 읽기

## Precondition
- https://the-internet.herokuapp.com/tinymce 접속 상태

## Steps
1. TinyMCE 에디터 iframe으로 전환
2. 에디터 본문의 기본 텍스트 읽기

## Expected
- 에디터에 "Your content goes here." 기본 텍스트가 표시된다
