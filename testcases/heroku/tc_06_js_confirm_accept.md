# JavaScript Confirm 수락

## Precondition
- https://the-internet.herokuapp.com/javascript_alerts 접속 상태

## Steps
1. "Click for JS Confirm" 버튼(button:has-text("Click for JS Confirm"))을 클릭한다
2. Confirm 대화상자를 수락(accept)한다

## Expected
- Confirm 대화상자가 "I am a JS Confirm" 텍스트를 포함한다
- 수락 후 결과 메시지(#result)가 "You clicked: Ok" 텍스트를 표시한다
