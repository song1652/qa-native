# Exit Intent 마우스 이탈 시 모달 표시

## Precondition
- https://the-internet.herokuapp.com/exit_intent 접속 상태

## Steps
1. 페이지 로드 확인
2. 마우스를 뷰포트 상단 밖으로 이동 (mouseout 이벤트 트리거)
3. 모달 다이얼로그 표시 확인

## Expected
- 마우스 이탈 시 모달 다이얼로그가 표시된다
- 모달 제목에 "This is a modal window" 텍스트가 포함된다
