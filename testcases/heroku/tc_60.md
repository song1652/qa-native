# 새 창 열기 후 원래 창 복귀

## Precondition
- https://the-internet.herokuapp.com/windows 접속 상태

## Steps
1. "Click Here" 링크 클릭
2. 새 창/탭으로 전환
3. 새 창 텍스트 확인
4. 원래 창으로 복귀
5. 원래 페이지 제목 확인

## Expected
- 원래 창으로 복귀 후 "Opening a new window" 제목이 표시된다
