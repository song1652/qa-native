# Entry Ad 모달 닫기

## Precondition
- https://the-internet.herokuapp.com/entry_ad 접속 상태

## Steps
1. 모달 다이얼로그 표시 대기
2. 모달의 "Close" 버튼 클릭
3. 모달 상태 확인

## Expected
- Close 버튼 클릭 후 모달이 사라진다
- 페이지 본문 콘텐츠가 표시된다
