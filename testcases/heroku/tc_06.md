# 요소 추가 및 삭제

## Precondition
- https://the-internet.herokuapp.com/add_remove_elements/ 접속 상태

## Steps
1. "Add Element" 버튼 클릭 3회
2. 추가된 "Delete" 버튼 개수 확인
3. "Delete" 버튼 1개 클릭

## Expected
- Add Element 3회 클릭 후 Delete 버튼이 3개 표시된다
- Delete 1개 클릭 후 Delete 버튼이 2개 남는다
