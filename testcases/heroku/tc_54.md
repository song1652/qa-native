# jQuery UI 메뉴 클릭 동작

## Precondition
- https://the-internet.herokuapp.com/jqueryui/menu 접속 상태

## Steps
1. "Enabled" 메뉴 항목에 호버
2. 서브메뉴 표시 대기
3. "Downloads" 서브메뉴 항목에 호버
4. 하위 메뉴 항목 클릭

## Expected
- 메뉴 항목 클릭이 정상 동작한다
- 파일 다운로드 또는 페이지 이동이 발생한다
