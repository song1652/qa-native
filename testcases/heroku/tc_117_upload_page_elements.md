# 업로드 페이지 요소 확인

## Precondition
- https://the-internet.herokuapp.com/upload 접속 상태

## Steps
1. 파일 선택 input 요소 존재 확인
2. "Upload" 버튼 존재 확인
3. 드래그 앤 드롭 영역 존재 확인

## Expected
- 파일 선택 input(#file-upload)이 존재한다
- Upload 버튼(#file-submit)이 존재한다
- 드래그 앤 드롭 영역(#drag-drop-upload)이 존재한다
