# 잘못된 계정 로그인 실패

## Precondition
- https://the-internet.herokuapp.com/login 접속 상태

## Steps
1. Username 필드에 test_data[invalid_user].username 입력
2. Password 필드에 test_data[invalid_user].password 입력
3. Login 버튼 클릭

## Expected
- /login 페이지에 머무른다
- "Your username is invalid!" 에러 메시지가 표시된다
