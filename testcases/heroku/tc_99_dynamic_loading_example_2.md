# Dynamic Loading Example 2 렌더링 후 표시

## Precondition
- https://the-internet.herokuapp.com/dynamic_loading/2 접속 상태

## Steps
1. "Start" 버튼 클릭
2. 로딩 완료 대기
3. 표시된 텍스트 확인

## Expected
- 로딩 후 "Hello World!" 텍스트가 표시된다
- 요소가 DOM에 새로 렌더링된다
