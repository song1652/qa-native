# jQuery UI 서브메뉴 호버 표시

## Precondition
- https://the-internet.herokuapp.com/jqueryui/menu 접속 상태

## Steps
1. "Enabled" 메뉴 항목에 마우스 호버
2. 서브메뉴 표시 대기
3. 서브메뉴 항목 확인

## Expected
- 호버 시 서브메뉴가 표시된다
- 서브메뉴에 "Downloads" 항목이 존재한다
