# 파일 다운로드 링크 확인

## Precondition
- https://the-internet.herokuapp.com/download 접속 상태

## Steps
1. 다운로드 가능한 파일 링크 목록 확인
2. 각 링크의 href 속성 확인

## Expected
- 다운로드 파일 링크가 최소 1개 존재한다
- 각 링크에 유효한 href가 설정되어 있다
