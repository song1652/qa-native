# 체크박스 초기 상태 확인

## Precondition
- https://the-internet.herokuapp.com/checkboxes 접속 상태

## Steps
1. 체크박스 요소 수 확인
2. 각 체크박스의 체크 상태 확인

## Expected
- 체크박스가 2개 존재한다
- 첫 번째 체크박스는 해제 상태이다
- 두 번째 체크박스는 체크 상태이다
