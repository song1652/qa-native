# Dynamic Controls 체크박스 제거 후 재추가

## Precondition
- https://the-internet.herokuapp.com/dynamic_controls 접속 상태

## Steps
1. 체크박스 존재 확인
2. "Remove" 버튼 클릭
3. 체크박스 사라짐 대기
4. "Add" 버튼 클릭
5. 체크박스 재표시 대기

## Expected
- "Remove" 클릭 후 체크박스가 사라진다
- "Add" 클릭 후 체크박스가 다시 표시된다
