# 동적 콘텐츠 새로고침 후 변경 확인

## Precondition
- https://the-internet.herokuapp.com/dynamic_content 접속 상태

## Steps
1. 첫 번째 콘텐츠 행의 텍스트 읽기
2. 페이지 새로고침
3. 첫 번째 콘텐츠 행의 텍스트 다시 읽기

## Expected
- 새로고침 후 콘텐츠가 변경될 수 있다
- 페이지 구조(3개 행)는 유지된다
