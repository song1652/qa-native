# JS Alert 텍스트 확인

## Precondition
- https://the-internet.herokuapp.com/javascript_alerts 접속 상태

## Steps
1. "Click for JS Alert" 버튼 클릭
2. Alert 다이얼로그의 메시지 텍스트 확인
3. OK 클릭

## Expected
- Alert 메시지가 "I am a JS Alert" 이다
- OK 클릭 후 결과에 "You successfully clicked an alert" 표시된다
