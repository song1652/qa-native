# Dynamic Controls 입력 필드 활성화 후 비활성화

## Precondition
- https://the-internet.herokuapp.com/dynamic_controls 접속 상태

## Steps
1. 텍스트 입력 필드가 비활성화 상태인지 확인
2. "Enable" 버튼 클릭
3. 입력 필드 활성화 대기
4. "Disable" 버튼 클릭
5. 입력 필드 비활성화 대기

## Expected
- "Enable" 클릭 후 입력 필드가 활성화된다
- "Disable" 클릭 후 입력 필드가 비활성화된다
