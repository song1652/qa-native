# Dynamic Loading 로딩 인디케이터 대기

## Precondition
- https://the-internet.herokuapp.com/dynamic_loading/1 접속 상태

## Steps
1. "Start" 버튼 클릭
2. 로딩 인디케이터 표시 확인
3. 로딩 인디케이터 사라짐 대기
4. 결과 텍스트 확인

## Expected
- "Start" 클릭 후 로딩 바가 표시된다
- 로딩 완료 후 로딩 바가 사라진다
- "Hello World!" 텍스트가 표시된다
