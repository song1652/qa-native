# 플로팅 메뉴 스크롤 후 고정 확인

## Precondition
- https://the-internet.herokuapp.com/floating_menu 접속 상태

## Steps
1. 페이지 하단으로 스크롤
2. 플로팅 메뉴 가시성 확인
3. 메뉴가 화면에 고정되어 있는지 확인

## Expected
- 스크롤 후에도 플로팅 메뉴가 화면에 표시된다
- 메뉴의 position이 fixed이다
