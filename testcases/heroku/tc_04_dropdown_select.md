# 드롭다운 옵션 선택

## Precondition
- https://the-internet.herokuapp.com/dropdown 접속 상태

## Steps
1. 드롭다운(#dropdown)에서 "Option 1"을 선택한다
2. 선택된 값을 확인한다
3. 드롭다운(#dropdown)에서 "Option 2"를 선택한다
4. 선택된 값을 확인한다

## Expected
- "Option 1" 선택 후 드롭다운의 selected value가 "1"이다
- "Option 2" 선택 후 드롭다운의 selected value가 "2"이다
