# JS Confirm 수락

## Precondition
- https://the-internet.herokuapp.com/javascript_alerts 접속 상태

## Steps
1. "Click for JS Confirm" 버튼 클릭
2. Confirm 다이얼로그에서 OK 클릭

## Expected
- Confirm 다이얼로그가 표시된다
- 결과 텍스트에 "You clicked: Ok" 표시된다
