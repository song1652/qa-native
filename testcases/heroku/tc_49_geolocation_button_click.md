# Geolocation 위치 정보 버튼 클릭

## Precondition
- https://the-internet.herokuapp.com/geolocation 접속 상태

## Steps
1. "Where am I?" 버튼 확인
2. 버튼 클릭
3. 위치 정보 결과 표시 대기

## Expected
- 버튼 클릭 후 위도(latitude)와 경도(longitude) 정보가 표시된다
