# JS Alert 확인

## Precondition
- https://the-internet.herokuapp.com/javascript_alerts 접속 상태

## Steps
1. "Click for JS Alert" 버튼 클릭
2. Alert 다이얼로그에서 OK 클릭

## Expected
- Alert 다이얼로그가 표시된다
- 결과 텍스트에 "You successfully clicked an alert" 표시된다
