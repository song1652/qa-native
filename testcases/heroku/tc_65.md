# 리다이렉트 최종 URL 확인

## Precondition
- https://the-internet.herokuapp.com/redirector 접속 상태

## Steps
1. "here" 리다이렉트 링크 클릭
2. 페이지 이동 완료 대기
3. 현재 URL 확인

## Expected
- 최종 URL이 "https://the-internet.herokuapp.com/status_codes"이다
- "Status Codes" 페이지 제목이 표시된다
