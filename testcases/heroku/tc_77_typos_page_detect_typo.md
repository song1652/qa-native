# Typos 페이지 오타 감지

## Precondition
- https://the-internet.herokuapp.com/typos 접속 상태

## Steps
1. 두 번째 단락 텍스트 읽기
2. 오타 여부 확인

## Expected
- 텍스트에 오타가 존재할 수 있다 (의도적 오류 페이지)
- 페이지가 정상적으로 로드된다
