# Sortable Tab Switch

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/sortable 페이지로 이동
2. "List" 탭 클릭하여 리스트 레이아웃 확인
3. "Grid" 탭 클릭하여 그리드 레이아웃 확인

## Expected
- "List" 탭에서 세로 목록 레이아웃이 표시됨
- "Grid" 탭에서 격자 레이아웃이 표시됨
