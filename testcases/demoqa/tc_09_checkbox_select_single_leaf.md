# Select Only Notes Leaf Checkbox

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/checkbox 접속
2. Expand All 버튼(+) 클릭하여 전체 트리 펼침
3. "Notes" 체크박스만 클릭

## Expected
- "Notes" 항목만 선택됨
- 페이지 하단 result 영역에 "notes" 텍스트만 표시됨
- 다른 항목은 선택되지 않음
