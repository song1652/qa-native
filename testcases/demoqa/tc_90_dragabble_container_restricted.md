# Dragabble Container Restricted

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/dragabble 페이지로 이동
2. "Container Restricted" 탭 클릭
3. 컨테이너 내 드래그 가능한 요소를 컨테이너 경계 밖으로 드래그 시도

## Expected
- 요소가 컨테이너 영역 밖으로 나가지 않음
- 컨테이너 내부에서만 이동 가능
