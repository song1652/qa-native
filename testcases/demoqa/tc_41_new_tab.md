# New Tab Opens with Sample Page Text

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/browser-windows 접속
2. "New Tab" 버튼 클릭
3. 새로 열린 탭으로 전환

## Expected
- 새 탭이 열림
- 새 탭에 "This is a sample page" 텍스트가 표시됨
