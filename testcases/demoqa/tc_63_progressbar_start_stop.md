# Progress Bar Starts and Stops at Intermediate Value

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/progress-bar 접속
2. Start 버튼 클릭
3. 일정 시간 대기
4. Stop 버튼 클릭

## Expected
- 프로그레스 바가 증가하다 멈춤
- 멈춘 값이 0~100 사이임
