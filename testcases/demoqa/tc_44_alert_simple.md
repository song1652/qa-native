# Simple Alert Can Be Accepted

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/alerts 접속
2. "Click Button to see alert" 버튼 클릭
3. 나타난 alert를 OK로 닫기

## Expected
- alert 대화상자가 발생함
- OK 버튼으로 alert를 닫을 수 있음
