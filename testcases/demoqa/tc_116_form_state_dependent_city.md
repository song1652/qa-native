# Practice Form State Dependent City Options

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/automation-practice-form 페이지로 이동
2. 하단 "State" 드롭다운 클릭
3. "Uttar Pradesh" 선택
4. "City" 드롭다운 클릭하여 옵션 목록 확인

## Expected
- City 드롭다운에 Agra, Lucknow, Merrut 옵션만 표시됨 (Uttar Pradesh 소속 도시)
