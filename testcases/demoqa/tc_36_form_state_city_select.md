# Form State City Dropdown Selection

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/automation-practice-form 접속
2. State 드롭다운 클릭
3. "NCR" 옵션 선택
4. City 드롭다운 클릭
5. "Delhi" 옵션 선택

## Expected
- State 드롭다운에 "NCR"이 표시됨
- City 드롭다운에 "Delhi"가 표시됨
