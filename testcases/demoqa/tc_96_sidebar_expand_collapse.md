# Sidebar Expand Collapse

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/elements 페이지로 이동
2. 사이드바에서 "Widgets" 그룹 헤더 클릭하여 펼치기
3. 하위 메뉴 표시 확인
4. 다시 "Widgets" 그룹 헤더 클릭하여 접기
5. 하위 메뉴 숨김 확인

## Expected
- 그룹 헤더 클릭 시 하위 메뉴가 펼쳐짐
- 다시 클릭 시 하위 메뉴가 접힘
