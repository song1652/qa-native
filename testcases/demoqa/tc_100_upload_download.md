# Upload Download

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/upload-download 페이지로 이동
2. "Download" 버튼 클릭
3. 파일 다운로드 시작 확인
4. Upload 영역에 파일 업로드
5. 업로드된 파일명 표시 확인

## Expected
- 다운로드 버튼 클릭 시 파일 다운로드가 시작됨
- 파일 업로드 후 업로드된 파일명이 페이지에 표시됨
