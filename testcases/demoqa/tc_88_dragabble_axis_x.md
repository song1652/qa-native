# Dragabble Axis X

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/dragabble 페이지로 이동
2. "Axis Restricted" 탭 클릭
3. "Only X" 요소를 드래그하여 수평 이동

## Expected
- 요소가 X축으로만 이동함
- Y축 위치는 고정된 채 변경되지 않음
