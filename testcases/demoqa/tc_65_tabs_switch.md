# Tabs Switch Content on Click

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/tabs 접속
2. "Origin" 탭 클릭
3. "Origin" 탭 내용 확인
4. "Use" 탭 클릭
5. "Use" 탭 내용 확인

## Expected
- "Origin" 탭 클릭 시 해당 내용이 표시됨
- "Use" 탭 클릭 시 해당 탭의 내용으로 전환되어 표시됨
