# Auto Complete Multi Color Selection Shows Tags

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/auto-complete 접속
2. "Type multiple color names" 필드에 "Re" 입력
3. 드롭다운에서 "Red" 선택
4. 동일 필드에 "Bl" 입력
5. 드롭다운에서 "Blue" 선택

## Expected
- "Red" 태그와 "Blue" 태그 2개가 필드에 표시됨
