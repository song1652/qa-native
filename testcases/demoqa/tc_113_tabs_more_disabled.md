# Tabs More Tab Disabled

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/tabs 페이지로 이동
2. 탭 목록에서 "More" 탭 확인

## Expected
- "More" 탭이 disabled 상태로 표시됨
- "More" 탭 클릭 시 아무런 동작도 발생하지 않음 (탭 전환 불가)
