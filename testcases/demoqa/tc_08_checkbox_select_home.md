# Select Home Checkbox to Select All Children

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/checkbox 접속
2. "Home" 체크박스 클릭

## Expected
- Home 및 모든 하위 항목이 선택(체크) 상태가 됨
- 페이지 하단 result 영역에 선택된 항목 텍스트가 표시됨 (home, desktop, notes, commands, documents 등)
