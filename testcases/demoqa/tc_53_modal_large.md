# Large Modal Opens and Can Be Closed

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/modal-dialogs 접속
2. "Large modal" 버튼 클릭
3. 모달 내용 확인
4. Close 버튼 클릭

## Expected
- 큰 모달이 표시됨
- 모달 내에 긴 텍스트 내용이 표시됨
- Close 버튼으로 모달을 닫을 수 있음
