# Dynamic Properties Text With Random Id

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/dynamic-properties 페이지로 이동
2. "This text has random Id" 텍스트가 표시된 요소 확인
3. 해당 요소의 id 속성 값 확인

## Expected
- "This text has random Id" 텍스트가 visible 상태로 표시됨
- 해당 요소에 id 속성이 존재하며 비어있지 않음
