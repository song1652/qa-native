# Checkbox Expand Single Node

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/checkbox 페이지로 이동
2. "Home" 노드 옆의 화살표(토글) 아이콘 클릭하여 한 단계만 펼치기

## Expected
- "Desktop", "Documents", "Downloads" 노드가 표시됨
- 각 하위 노드의 자식 노드는 접힌 상태로 표시됨 (더 하위 노드 미표시)
