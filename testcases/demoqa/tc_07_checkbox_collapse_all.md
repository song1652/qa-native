# Collapse All Checkbox Tree Nodes After Expanding

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/checkbox 접속
2. Expand All 버튼(+) 클릭하여 전체 트리 펼침
3. Collapse All 버튼(-) 클릭

## Expected
- 루트 "Home" 노드만 표시됨
- 하위 노드(Desktop, Documents, Downloads 등)가 모두 숨겨짐
