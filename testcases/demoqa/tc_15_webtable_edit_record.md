# Edit Existing Record in Web Table

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/webtables 접속
2. 첫 번째 행의 Edit 버튼(연필 아이콘) 클릭
3. First Name 필드를 "UpdatedName"으로 변경
4. Submit 버튼 클릭

## Expected
- 모달이 닫힘
- 테이블의 첫 번째 행 First Name 값이 "UpdatedName"으로 변경되어 표시됨
