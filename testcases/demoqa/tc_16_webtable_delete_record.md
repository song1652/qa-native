# Delete Record From Web Table

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/webtables 접속
2. 첫 번째 행에서 표시되는 이름 값을 메모
3. 첫 번째 행의 Delete 버튼(삭제 아이콘) 클릭

## Expected
- 클릭한 행이 테이블에서 삭제됨
- 이전에 첫 번째 행이었던 데이터가 테이블에 더 이상 존재하지 않음
