# Web Table Sort By First Name

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/webtables 페이지로 이동
2. "First Name" 컬럼 헤더 클릭
3. 테이블 첫 번째 행의 First Name 값 확인
4. "First Name" 컬럼 헤더 다시 클릭
5. 테이블 첫 번째 행의 First Name 값 재확인

## Expected
- 첫 번째 클릭 후 이름 기준 오름차순 정렬됨
- 두 번째 클릭 후 이름 기준 내림차순 정렬됨
