# Links Home Link New Tab

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/links 페이지로 이동
2. "Home" 링크를 클릭 (새 탭으로 열림)
3. 새 탭이 열릴 때까지 대기

## Expected
- 새 탭이 열리며 demoqa.com 홈페이지가 로드됨
- 새 탭의 URL이 https://demoqa.com/ 임
