# Practice Form Subject Remove Tag

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/automation-practice-form 페이지로 이동
2. "Subjects" 입력란에 "Maths" 입력 후 Enter 또는 자동완성 선택으로 태그 추가
3. 추가된 "Maths" 태그 옆의 X 버튼 클릭

## Expected
- "Maths" 태그가 Subjects 입력란에서 제거됨
- Subjects 입력란이 빈 상태로 표시
