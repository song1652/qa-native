# Submit Text Box With No Input

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/text-box 접속
2. 아무 필드도 입력하지 않음
3. Submit 버튼 클릭

## Expected
- 하단 출력 영역(#output)이 표시되지 않거나 비어있음
- 어떠한 값도 출력 영역에 표시되지 않음
