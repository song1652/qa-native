# Sidebar Navigation

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/elements 페이지로 이동
2. 사이드바에서 "Check Box" 클릭
3. 해당 페이지 내용 확인
4. 사이드바에서 "Radio Button" 클릭
5. 해당 페이지 내용 확인
6. 사이드바에서 "Web Tables" 클릭
7. 해당 페이지 내용 확인

## Expected
- 각 사이드바 항목 클릭 시 해당 페이지 내용이 표시됨
- URL이 각 항목에 맞게 변경됨
