# Selectable Multiple List Items

## Precondition
- https://demoqa.com 접속 상태

## Steps
1. https://demoqa.com/selectable 페이지로 이동
2. "List" 탭이 활성화된 상태 확인
3. 리스트에서 첫 번째 항목 클릭
4. 두 번째 항목 클릭
5. 세 번째 항목 클릭

## Expected
- 클릭한 세 항목 모두 active(선택됨) 상태로 표시 (배경색 변경)
